📋 ¿Cómo obtener la lista de productos?
1. Abre el ERP en Chrome y presiona ⌘ + Option + J (o clic derecho → Inspeccionar → Consola)
2. Escribe allow pasting y presiona Enter (es normal que salga un error)
3. Ahora pega este comando y presiona Enter:
copy(JSON.stringify(getProductos()))
4. Vuelve aquí y pega el resultado (⌘V) en el cuadro de abajo
